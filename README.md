# Console-Based Movie Recommender

A Python console application that creates user accounts, validates user input,
filters movies based on age restrictions, and allows users to search movies by genre.

## Features

- Email validation using allowed domains
- Password validation (8-15 characters)
- Age-based movie filtering
- Genre-based movie search
- Built using Python fundamentals (Loops, Functions, Dictionaries, If-Elif-Else Statements)

## Technologies Used
- Python 3

## How to Run
1. Clone the repository
2. Run 'selfproject.py' using Python 3
3. Follow the prompts in the Console Log/Terminal
